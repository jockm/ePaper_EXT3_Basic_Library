Detailed Description

Class for Pervasive Displays iTC BWR screens.

Screen controllers

  • LCD: propietary, SPI
Note
Contrary to eTC, only one frame-buffer is required for iTC BWR.


All commands work on the frame-buffer, to be displayed on screen with flush()
